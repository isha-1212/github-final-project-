# Simple Interest Calculator

This project calculates simple interest based on principal amount, rate of interest, and time period.

## Formula
Simple Interest = (Principal × Rate × Time) / 100

## Usage
Run the shell script and enter:
- Principal amount
- Rate of interest
- Time period

The script will display the calculated simple interest.

## Example
Principal: 1000  
Rate: 5  
Time: 2  

Simple Interest = 100
